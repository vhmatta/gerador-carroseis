# Gerador de Capas LinkedIn - Parcele Aqui

**Versão 1.0 - 2026**  
Desenvolvido por **Potencial Tecnologia**

---

## 📋 Sobre o App

Aplicativo profissional para gerar capas personalizadas para posts do LinkedIn no formato 1200x675px, otimizado para a identidade visual da **Parcele Aqui**.

---

## ✨ Funcionalidades

### 🎨 Editor Individual
- Edição em tempo real de uma capa por vez
- Preview imediato das alterações
- Campos editáveis:
  - Número da edição (ex: #49)
  - Título principal (até 3 linhas)
  - Legenda linha 1 (sobre a imagem)
  - Legenda linha 2 (negrito)
  - Seletor de 20 ícones profissionais
  - Busca de imagens profissionais
- Botão para gerar e baixar PNG individual

### 📦 Gerador em Lote
- Gere até 60 capas de uma vez
- Campo de prompt para preenchimento rápido
- Configuração de grupos de imagens/ícones
- Opções: 10, 20, 30, 40, 50 ou 60 capas por grupo
- Navegação entre capas geradas
- Download em massa de todas as imagens

---

## 🚀 Como Usar

### Editor Individual

1. **Preencha os campos** na seção de configuração
2. **Veja o preview** da capa em tempo real acima
3. **Ajuste conforme necessário**
4. **Clique em "Gerar Imagem"** para baixar o PNG

### Gerador em Lote

1. **Cole os títulos** - um por linha no campo "Títulos"
2. **OU use o Prompt** no formato:
   ```
   Título da capa 1 | CreditCard | https://imagem.com/1.jpg
   Título da capa 2 | Wallet | https://imagem.com/2.jpg
   ```
3. **Configure imagens e ícones** para cada grupo
4. **Clique em "Gerar Capas"**
5. **Navegue** entre as capas usando os botões ← →
6. **Clique em "Gerar Imagens"** para baixar todas

## ⚙️ Especificações Técnicas

### Layout das Capas
- **Dimensões:** 1200x675 pixels
- **Formato:** PNG de alta qualidade (scale 2x)
- **Fonte dos títulos:** Kufam (Original Type)
- **Cores:** 
  - Amarelo: #FFC528
  - Amarelo claro: #FFE8A4
  - Preto: #1a1a1a

### Elementos Customizáveis
- Número da edição (canto superior direito)
- Título principal (até 3 linhas, lado esquerdo)
- Ícone (20 opções disponíveis)
- Legenda sobre a foto (2 linhas opcionais)
- Imagem de fundo (500x470px, canto arredondado)

---

## 🎯 Dicas de Uso

### Títulos Efetivos
- **Máximo 3 linhas** para melhor leitura
- Use **frases curtas e diretas**
- Evite palavras muito longas

### Escolha de Imagens
- **Prefira rostos** quando possível (maior engajamento)
- **Evite imagens muito escuras** (legenda fica difícil de ler)
- **Teste o contraste** da legenda com a imagem

### Ícones
- Use ícones **relacionados ao tema** do post
- **CreditCard, Wallet** - para conteúdos financeiros
- **TrendingUp, BarChart** - para resultados e métricas
- **Users, Target** - para estratégia e público

---

## 🛠️ Resolução de Problemas

### Imagem não aparece na capa
- Verifique se a URL é válida
- Use imagens do seletor integrado
- Certifique-se que a URL começa com `https://`

### Download não funciona
- Aguarde o preview carregar completamente
- Verifique as configurações do navegador
- Permita downloads múltiplos para o gerador em lote

### Fontes não aparecem corretamente
- Aguarde alguns segundos após abrir o app
- Recarregue a página se necessário
- As fontes são carregadas automaticamente

## 📝 Formato do Prompt

Para preencher rapidamente no Gerador em Lote:

```
Título completo do post | NomeDoIcone | URLDaImagem
```

**Exemplo:**
```
Mercado B2C: estratégias de parcelamento | CreditCard | https://images.unsplash.com/photo-123
Como otimizar o fluxo de caixa | Wallet
Tendências em pagamentos digitais | TrendingUp
```

**Ícones disponíveis:**
- CreditCard, Wallet, TrendingUp, DollarSign
- Sparkles, Target, BarChart3, PieChart
- ShoppingCart, Store, Zap, TrendingDown
- LayoutDashboard, Megaphone, Lightbulb, Rocket
- Users, ShieldCheck, Lock, Unlock
